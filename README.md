# GoogleApis

Consuma las APIs de Google con esta DLL para VBA. Este repositorio proporciona una DLL en C# que permite consumir las APIs de Google Drive y Google Sheets.

## Uso

Para utilizar esta DLL en tu proyecto de VBA, sigue los siguientes pasos:

1. Descarga la DLL desde la sección de [releases](https://github.com/tu-usuario/GoogleApis/releases) de este repositorio.
2. Agrega la DLL a tu proyecto de VBA. Puedes hacerlo siguiendo estos pasos:
   - Abre tu proyecto de VBA en el editor de Visual Basic.
   - Haz clic en "Herramientas" en la barra de menú y selecciona "Referencias".
   - En la ventana de referencias, haz clic en "Examinar" y selecciona la DLL descargada.
   - Haz clic en "Aceptar" para agregar la referencia a la DLL.
3. Ahora puedes utilizar las APIs de Google Drive y Google Sheets en tu proyecto de VBA. Asegúrate de importar los espacios de nombres necesarios y seguir la documentación de las APIs para realizar las operaciones deseadas.
